5. 데이터 초기화

`조별삭제`는 현재 선택된 조 데이터만 지웁니다.
`전체삭제`는 모든 조 데이터를 지우고 화면 상태를 대기로 초기화합니다.
삭제/초기화는 두 번 확인하고, 내려받기는 엑셀에서 열 수 있는 CSV 백업 파일로 저장합니다.

6. 운영 단축키

1~9, 0선택 조를 미리보기로 불러옵니다. `0`은 10조입니다.
Enter현재 선택한 조를 1페이지부터 송출합니다.
O현재 편집 중인 오프닝 슬라이드부터 송출합니다.
← / →현재 송출 중인 발표 페이지 또는 오프닝 슬라이드를 이전/다음 장으로 넘깁니다.
Space타이머를 시작하거나 일시정지합니다.
R입력한 분 수 기준으로 타이머를 초기화합니다.
W현재 대기화면 문구로 대기화면으로 전환합니다.
F피날레 연출을 시작합니다.

입력창, 텍스트 영역, 셀렉트 박스에 포커스가 있을 때는 단축키가 동작하지 않습니다. 실수 방지를 위해 `Ctrl`, `Cmd`, `Alt` 조합과 길게 누른 반복 입력도 무시합니다.
